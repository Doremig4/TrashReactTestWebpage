# BrainSpark: 인터랙티브 퀴즈 플랫폼

React와 Modern Web Tech를 활용하여 개발된 반응형 퀴즈 및 게임 웹 애플리케이션입니다.

## 기술 스택 및 선정 이유(Technical Stack)

이 프로젝트는 최신 웹 개발 트렌드와 생산성을 고려하여 다음과 같은 기술들을 선정했습니다.

### 1. **React (`react`)**
- **선정 이유**: 컴포넌트 기반 아키텍처를 통해 UI를 재사용 가능한 단위로 모듈화하기 위해 선택했습니다.
- 방대한 생태계와 커뮤니티 지원으로 문제 해결이 용이하며, 단방향 데이터 흐름을 통해 앱의 상태를 예측 가능하게 관리할 수 있습니다.

### 2. **Vite (`create-vite`)**
- **선정 이유**: 기존 CRA(Create React App) 대비 압도적으로 빠른 개발 서버 구동 속도와 HMR(Hot Module Replacement) 기능을 제공합니다.
- `esbuild`를 기반으로 하여 빌드 속도가 매우 빠르며, 복잡한 설정 없이도 최신 모던 웹 개발 환경을 즉시 구축할 수 있어 채택했습니다.

### 3. **Framer Motion (`framer-motion`)**
- **선정 이유**: React 환경에서 애니메이션을 선언적으로(declarative) 쉽게 구현하기 위해 사용했습니다.
- 단순한 CSS 트랜지션보다 훨씬 복잡한 제스처, 레이아웃 애니메이션, 페이지 전환 효과 등을 직관적인 API로 구현할 수 있어 사용자 경험(UX)을 극대화했습니다.

### 4. **Lucide React (`lucide-react`)**
- **선정 이유**: 일관된 디자인 언어를 가진 가볍고 깔끔한 SVG 아이콘 라이브러리입니다.
- 런타임 크기가 작고, 커스터마이징이 용이하여 모던한 UI 디자인에 적합합니다.

### 5. **Prop Types (`prop-types`)**
- **선정 이유**: TypeScript를 사용하지 않는 JavaScript 환경에서 컴포넌트 간 전달되는 데이터(Props)의 타입을 런타임에 검증하기 위해 사용했습니다.
- 개발 단계에서 잘못된 데이터 전달로 인한 오류를 사전에 방지하여 코드의 안정성을 높여줍니다.

## 🚀 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
